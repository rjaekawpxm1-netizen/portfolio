📊 청주시 교통사고 분석 & AI 해석 자동화 프로젝트
1. 프로젝트 개요

본 프로젝트는 청주시 교통사고 현황을 전국 평균과 비교 분석하고,
그 결과를 Power BI 시각화 + LLM 기반 자동 해석으로 정리하는 것을 목표로 한다.
단순 지표 나열이 아닌, 비교·구성·치명도 관점에서 사고 특성을 해석하는 데 초점을 두었다.

2. 사용 데이터

전국 시군구 교통사고 요약 데이터

청주시 교통사고 요약 데이터

주요 변수

사고건수, 사망자수, 중상자수, 경상자수, 부상신고자수

사상자 합계

사상자 대비 사망비율

사고 1건당 사상자·중상자·사망자

3. 분석 및 시각화 (Power BI)

다음과 같은 페이지로 구성하였다.

① 핵심 지표(KPI) 요약

사고건수, 사상자수, 사망자수, 사망비율 등 주요 수치 한눈에 확인

② 사망비율 비교

청주시 vs 전국 평균

사상자 대비 사망비율 기준으로 치명도 수준 비교

③ 사고 심각도 구성

사망·중상·경상·부상신고 비율 도넛 차트

청주시와 전국의 구성 구조 비교

④ 사고 1건당 피해 규모

사고 1건당 사상자·중상자·사망자 수 비교

절대 규모가 아닌 사고 단위 피해 수준 중심 해석

👉 모든 시각화에는 해석 캡션을 직접 추가하여,
그래프만 봐도 분석 결론이 자연스럽게 전달되도록 구성했다.

4. LLM 활용 (AI 해석 자동화)

Power BI 분석 결과를 기반으로,
Python과 LLM을 활용해 사람이 읽기 쉬운 해석 문장을 자동 생성했다.

🔹 자동 생성 산출물

metrics.json : 분석 지표 요약(JSON)

report.md : 보고서용 분석 문단

captions.json :

차트 캡션(short/medium/long)

발표용 멘트(10s/20s/40s)

👉 LLM은 분석을 대신하지 않고,
이미 계산된 수치를 해석·요약하는 역할로 제한해 신뢰성을 확보했다.

5. 주요 분석 결과 요약

청주시의 사상자 대비 사망비율은 전국 평균보다 낮은 수준이다.

청주시와 전국 모두 경상자 비중이 가장 높은 유사한 심각도 구조를 보인다.

사고 1건당 피해 규모 역시 청주시는 전국 평균 대비 낮거나 유사한 수준으로 나타났다.

6. 한계 및 유의점

본 분석은 집계 데이터 기반으로, 사고 원인이나 환경적 요인은 포함하지 않는다.

특정 정책 효과나 인과관계를 단정할 수 없다.

기간·도로 유형·시간대·기상 조건 등 세부 변수는 추가 데이터 확보 시 확장 가능하다.

Power BI를 활용한 정량적 비교 분석

Python + LLM을 결합한 해석 자동화 파이프라인 구현

데이터 분석 결과를 보고서·발표·시각화에 바로 활용 가능한 형태로 완성
