# 🛒 온라인 쇼핑몰 데이터 분석 (SQL + Python)

## 📌 1. 프로젝트 개요
이 프로젝트는 가상의 온라인 쇼핑몰 데이터를 기반으로  
**매출 분석 / 고객 분석 / 재구매 분석 / 채널 성과 분석**을 수행한 데이터 분석 프로젝트입니다.

전 데이터셋을 Faker 라이브러리로 직접 생성하고  
SQLite로 저장한 뒤 SQL과 Python(pandas, matplotlib)으로 분석했습니다.

---

## 📌 2. 사용 기술 스택

- **SQLite** – 관계형 데이터 저장/관리  
- **Python** – 데이터 생성 및 분석  
- **pandas** – 집계 및 분석  
- **matplotlib** – 시각화  
- **faker / tqdm** – 더미데이터 생성 및 진행률 표시  

---

## 📌 3. 데이터 구성

| 테이블 | 설명 |
|-------|------|
| customers | 고객 성별, 나이, 가입일 |
| products | 상품명, 카테고리, 가격 |
| orders | 주문일, 결제수단, 디바이스 |
| order_items | 주문별 상품 및 수량 |

데이터 규모:
- 고객 **2,000명**  
- 상품 **300개**  
- 주문 **5,000건**  
- 주문 상세 **10,000건**

---

## 📌 4. 주요 분석 내용

### ✔ 4-1. 월별 매출 분석
- 월별 매출 및 주문 수 집계  
- 매출 상승/하락 구간 파악  

### ✔ 4-2. 카테고리별 매출 분석
- 어떤 카테고리가 가장 수익이 높은지 분석  

### ✔ 4-3. 성별·연령대별 AOV 분석
- 핵심 고객군(20–40대)
- 고가 소비군(40대 여성)

### ✔ 4-4. 재구매율 분석
- 전체 고객 중 재구매 고객 비율 파악  
- 첫 구매 후 이탈 고객 비율 확인  

### ✔ 4-5. 디바이스·결제수단별 매출
- 모바일/앱 비중이 매우 높음  
- 간편결제(카카오페이/네이버페이) 강세
