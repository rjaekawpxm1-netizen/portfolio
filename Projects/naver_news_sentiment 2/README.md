뉴스 댓글 감정 분석


1. 프로젝트 개요

본 프로젝트는 뉴스 기사 댓글 데이터를 활용하여
댓글에 담긴 감정을 정량적으로 분석하고,
여론의 전반적인 분위기와 반응 패턴을 파악하는 것을 목표로 합니다.

텍스트 데이터를 단순 분류하는 것을 넘어,
비정형 데이터가 어떻게 분석 가능한 정보로 전환되는지에 초점을 두었습니다.

2. 문제 정의 (Problem Statement)

뉴스 댓글은 여론을 빠르게 파악할 수 있는 중요한 데이터이지만 다음과 같은 어려움이 있습니다.

텍스트 형태의 비정형 데이터

감정이 직관적이지만 정량화가 어려움

기사 내용과 댓글 반응 간의 분위기 차이를 파악하기 어려움

👉 본 프로젝트에서는
“뉴스 댓글에 담긴 감정을 데이터로 표현할 수 있을까?” 를 핵심 문제로 설정했습니다.

3. 데이터 설명

뉴스 기사 댓글 텍스트 데이터

주요 컬럼 예시:

댓글 내용

작성 시점

기사 카테고리 (정치, 사회, 경제 등)

4. 분석 접근 방식
4-1. 텍스트 전처리

불필요한 기호, 특수문자 제거

공백 및 중복 데이터 처리

분석에 불필요한 댓글 필터링

4-2. 감정 분석

텍스트 데이터를 벡터화

감정 분류 모델을 활용한 긍·부정 분석

댓글별 감정 점수 산출

5. 핵심 분석 내용
5-1. 댓글 감정 분포 분석

전체 댓글의 긍정 / 부정 비율 분석

기사별, 카테고리별 감정 분포 비교

5-2. 시간 흐름에 따른 감정 변화

댓글 작성 시점 기준 감정 변화 추적

특정 이슈 발생 시 감정 쏠림 현상 확인

6. 주요 인사이트

특정 뉴스 주제에서 부정 감정 댓글 비율이 높게 나타남

기사 카테고리에 따라 감정 분포가 명확히 달라짐

여론은 단일 감정보다 시간에 따라 변화하는 흐름으로 이해하는 것이 중요함

7. 문제 발생 및 해결 과정
문제 1. 댓글 텍스트의 노이즈 문제

문제 상황
댓글에는 의미 없는 반복 문자, 욕설, 단문 댓글이 다수 포함되어
분석 정확도를 저해했습니다.

해결 과정

최소 글자 수 기준 설정

불필요한 문자 및 패턴 제거

분석 대상 댓글 정제

해결 결과

감정 분석 모델의 안정성 향상

해석 가능한 결과 확보

문제 2. 감정 분석 결과 해석의 어려움

문제 상황
긍·부정 결과만으로는
왜 그런 결과가 나왔는지 설명하기 어려웠습니다.

해결 과정

기사 단위, 카테고리 단위로 감정 결과 재집계

단순 분류 결과를 맥락 기반 해석으로 확장

해결 결과

단일 댓글이 아닌 여론 흐름 중심 분석 가능
